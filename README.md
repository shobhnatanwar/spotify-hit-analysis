# SPOTIFY POPULARITY ANALYSIS

## OBJECTIVE
To identify the key factors that influence song popularity and artist success by analyzing track-level data, and to provide actionable insights on how artists can maximize performance in a highly competitive music landscape.

## TOOLS USED
1. Power BI – Interactive dashboard design and visualization  
2. DAX – Measures and calculated columns for analytical modeling  
3. Power Query – Data cleaning and transformation  
4. Excel – Initial data validation and preprocessing  

## KEY INSIGHTS
1. Only ~18% of songs achieve high popularity, indicating that success is highly selective and competitive.  
2. Increasing the number of releases does not improve success rates—artist output shows little to no correlation with average popularity.  
3. Track characteristics such as duration have minimal impact on popularity, with high- and low-performing songs showing similar averages (~3.5 minutes).  
4. Explicit content does not significantly influence performance, with nearly identical proportions in both high- and low-popularity groups.  
5. Genre contributes to variation in average popularity, but no genre consistently guarantees success.  

## FILES
1. Dataset – Spotify track-level data from Kaggle (popularity, duration, genre, artist, explicit flag)  
2. Power BI Dashboard – Multi-page interactive report (Executive Overview, Description, Artist Deep Dive)  
3. Images – Dashboard previews for GitHub documentation  

## OUTCOME
Delivered a data-driven analysis of music performance, enabling:
1. Better understanding of what differentiates successful songs  
2. Identification of ineffective assumptions (e.g., more output = more success)  
3. Strategic focus on high-impact releases rather than volume  

## BUSINESS IMPACT
This project demonstrates how data analytics can guide decision-making in the music industry by showing that popularity is not driven by basic track features alone, but likely by external factors such as promotion, audience reach, and platform exposure.
